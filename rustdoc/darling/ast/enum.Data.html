pub enum Data<V, F> {
    Enum(Vec<V, Global>),
    Struct(Fields<F>),
}
Expand description

A struct or enum body.

V is the type which receives any encountered variants, and F receives struct fields.

Variants

Enum(Vec<V, Global>)

Struct(Fields<F>)
